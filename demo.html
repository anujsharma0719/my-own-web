Astrology Mobile App – Demo

A sleek, cosmic-powered mobile app built for astrology lovers — modern UI, secure auth, and smooth experience.

Secure Login System

Users can securely log in using email/password or Google Sign-In.
Authentication is handled through Firebase to ensure safe and reliable user access.

  • Email & Google authentication
  • Secure Firebase backend

Home Dashboard

The home screen serves as the main dashboard, providing quick access to all astrology features in a clean and easy-to-navigate interface.

  • Feature-based navigation
  • Clean and user-friendly UI

User Profile Management

This screen allows users to view and manage their personal information, account details, and current subscription status from a single place.

  • Editable user profile
  • Subscription plan display

App Settings

The settings section provides options such as language selection, support contact, privacy policy, and terms for better user control.

  • Language & preferences
  • Privacy and support access
